# Pazarama-MovieApp
I developed this project for Pazarama bootcamp using OMDB API.

## Technologies I use
1. Protocol Oriented Programming
2. Generic Network Layer (URLSession)
3. MVVM
4. Programmatic UI Design
5. Dependency Injection
6. Delegate Pattern
7. KingFisher
8. NVActivityIndicator
9. Unit / UI Test
